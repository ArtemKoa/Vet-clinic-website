Данный проект написания веб сайта направлен на повторения основ веб-программирования. Созданный сайт должен был подходить для использования как на телефонах, так и на ПК. На сайте должно распологаться несколько страниц, с формой записи и обратной связью. На каждой странице информация представлена в надлежащем виде с изображениями для понятности услуг.
